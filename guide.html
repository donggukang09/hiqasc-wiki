📊 DOA (Dead on Arrival) 처리

DOA란?

제품 수령 즉시 불량이 확인된 제품으로, 수리 없이 교환 처리됩니다.

처리 절차

1. 제품 수령 시 외관 점검
2. 초기 작동 테스트 (통전 전)
3. DOA 확인 시 즉시 시스템에 등록
4. 사진 촬영 (제품, 포장, 문제 부위)
5. 워런티 클레임 자동 생성
6. 교체품 발송 대기

시스템 접근

• DOA 자동화 시스템 링크
• 시리얼 넘버 자동 매칭
• Excel 보고서 자동 생성

⏰ 초과근무 신청

신청 방법

1. 초과근무 관리 시스템 접속
2. 날짜 및 시간 입력
3. 사유 작성
4. 제출 (팀장 자동 알림)
5. 승인 대기 (이메일 알림 수신)

주의사항

• 주간 최대 12시간 제한
• 사전 승인 필수
• 긴급 상황 시 구두 승인 후 사후 등록 가능

📝 일일 보고서

작성 방법

1. Google Forms 접속 (매일 17:00 이전)
2. 수리 완료 대수 입력
3. 미완료 사유 선택
4. 특이사항 기재
5. 제출 → 자동으로 마스터 시트에 통합

보고 항목

• 수리 완료: ORIGINAL / VERTUO 구분
• 진행 중: 부품 대기 / 복잡한 수리
• DOA 발견: 대수 및 모델
• 특이사항: 새로운 증상, 부품 부족 등

📞 콜센터 연동 시스템

사용 방법

1. 콜센터 시스템 접속
2. 시리얼 넘버 또는 고객명 검색
3. 필터링된 요청 건 확인
4. 수리 상태 업데이트
5. 완료 시 자동으로 고객에게 알림

📦 부품 요청

요청 절차

1. 부품 사용 추적 시스템 접속
2. 필요 부품 코드 입력
3. 수량 및 긴급도 선택
4. 제출 → 자동으로 재고 팀에 전달
5. 입고 시 알림 수신

재고 확인

• 실시간 재고 대시보드 확인
• ORIGINAL / VERTUO 구분
• 부족 시 자동 알림

📈 실시간 재고 대시보드

기능

• 수리 대기 수량 실시간 확인
• ORIGINAL / VERTUO 분류
• 일일/주간 처리량 그래프
• 목표 대비 진행률
